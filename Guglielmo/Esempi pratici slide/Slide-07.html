HTML e CSS: Due mondi distinti. HTML = Struttura CSS = Stile Separazione Intelligente -> Migliora manutenzione, riuso del codice e chiarezza. Permette modifiche rapide senza toccare il contenuto. Regola CSS = Selettore + Dichiarazioni Il selettore indica a quali elementi HTML applicare lo stile. La dichiarazione comprende la coppia proprietà e valore. h1 { color: red; font-size: 32px; } Selettore -> Proprietà -> Valore. color -> colore del testo background-color -> colore di sfondo font-family -> tipo di carattere font-size -> dimensione del testo margin -> spazio esterno padding -> spazio interno border -> bordo dell'elemento display -> tipo di visualizzazione position -> posizionamento float -> allineamneto flottante

Ideale per pagine singole, prototipi veloci o quando si vuole testare rapidamente uno stile. Crea un .ccs file separato, lo colleghi con link nella sezione head dell'HTML.
