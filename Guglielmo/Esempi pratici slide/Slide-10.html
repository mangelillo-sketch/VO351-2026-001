È un linguaggio di scripting utilizzato nella programmazione di siti web lato client e pertanto non necessita di un server per il suo funzionamento. Javascript può essere incluso all'interno di una pagina HTML sia internamente che esternamente (con un file con estensione .js). Javascript è un linguaggio case sensitive, per questo scrivere una variabile o una parola chiave di sistema (come if, while, var, ecc.) in minuscolo o maiuscolo ha una enorme differenza.
