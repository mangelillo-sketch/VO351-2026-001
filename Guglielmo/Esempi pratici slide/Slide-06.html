= Sono i quattro pilastri dei form HTML.
"email" = Attiva la tastiera con @ e .com su mobile. Valida automaticamente il formato email e impedisce l'invio di dati non validi. "tel" = Mostra il tastierino numerico telefonico su smartphone, ottimizzato per inserire rapidamente numeri di telefono con caratteri speciali. "URL" = Tastiera con .com, .it e altri domini comuni. Valida che l'indirizzo inizi con http:// o https:// per garantire URL corretti. "number" = Input numerico con frecce su/giù per incrementare o decrementare. Perfetto per quantità, età o qualsiasi valore numerico. "date" = Apre un calendario nativo del browser per selezionare date facilmente, eliminando errori di formato e rendendo l'input intuitivo. required = Rende obbligatorio un campo. Il form non può essere inviato se il campo è vuoto. Essenziale per dati critici come nome, email o password. pattern = Definisce un'espressione regolare che il valore deve rispettare. Perfetto per codici fiscali, CAP, codici prodotto o qualsiasi formato specifico. min/max = Imposta valori minimi e massimi per numeri e date. Utile per età, quantità, periodi di prenotazione o qualsiasi range definito. minlength/maxlength = Controlla la lunghezza del testo inserito. Previene password troppo corte o commenti eccessivamente lunghi che potrebbero causare problemi. Table = Contenitore principale della tabella. Tr = Table Row, definisce una riga. Td = Table Data, una cella di dati. Th = Table Header, intestazione di colonna. Thead = Raggurppa le righe di intestazione. Tbody = Raggruppa le righe di contenuto. Description = Il riassunto della pagina che appare nei risultati di ricerca. Deve essere coinvolgente, unico per ogni pagina, e contenere parole chiave naturalmente integrate. Keywords = Sebbene meno influenti oggi, indicano i termini principali della pagina. Non esagerare: 5-10 parole chiave pertinenti e specifiche sono sufficienti. Viewport = Fondamentale per il responsive design. Controlla come la pagina si ridimensiona su dispositivi mobili, garantendo un'esperienza ottimale su ogni schermo. Corso HTML - Tutorial Completo
